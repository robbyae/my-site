CPU Gregorian Chant is a web application which procedurally generates medieval church plainsong and accompanying notation in the Gregorian style.

This project is a culmination of my passion for music, medieval history, software development, and pixel graphics. It is coded mostly in HTML, CSS, and JavaScript, but Python is used to generate the music. Exsurge.js renders the notation, and Tone.js renders the sound. The monks are original pixel art designed by me in Pixelorama.
