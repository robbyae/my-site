Original Wordpress Theme and Blog Design in PHP

Link to Blog

Using PHP and CSS, I designed an original theme to be used with Wordpress for the tech company Circlebox in Birmingham, Alabama. I was visually inspired by the preexisting "2020" theme, but was appalled by its atrocious user experience. Using a very barebones template, I created an original theme based on the 2020 theme but with an emphasis on exceptional user experience.
