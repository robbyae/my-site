This project is a demo for a concept to improve a viewer's understanding of any art piece. In this particular case, a mosaic from the Hagia Sophia is displayed, and its most important elements are hoverable. Upon clicking an element, information about that element is displayed on the page. This project is web-based and made with HTML, CSS, and Jquery for the web components and Adobe Illustrator for the hoverable SVG elements.
